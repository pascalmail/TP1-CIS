## Deskripsi Singkat
Aplikasi desktop yang dapat melakukan enkripsi dan dekripsi berkas menggunakan algoritma XTS-AES.
Tampilan GUI dibuat menggunakan Java Swing.

### Package aescalc
    Kelas-kelas yang diambil dari AESCalc.jar, dengan sedikit perubahan.

### Package cis.tp1
    Kelas XTSAES : Implementasi XTS-AES cipher untuk satu data unit.
    Kelas FileCipherXTSAES: Enkripsi dan dekripsi file menggunakan XTS-AES.
